## 🚀 Getting Started (Local Setup)

### Prerequisites
- Node.js 18+
- PostgreSQL
- Docker (optional)

### Backend
```bash
cd backend
cp .env.example .env
npm install
npx prisma migrate dev
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Frontend runs on `http://localhost:3000`
Backend runs on `http://localhost:5000`
